## API Endpoints

| EndPoint              | Functionality                  |
| --------------------- | ------------------------------ |
| GET /orders           | Get all the orders.            |
| GET /orders/<orderId> | Fetch a specific order         |
| POST /orders          | Place a new order.             |
| PUT /orders/<orderId> | Update the status of an order. |
